Politique de protection des données personnelles

1. Responsable du traitement

Le responsable du traitement est LOGIX FORMA, joignable à l’adresse : logix.forma@gmail.com.

2. Données collectées

Selon les formulaires utilisés sur le site, LOGIX FORMA peut collecter les données suivantes :

  • prénom ;
  • nom ;
  • adresse email ;
  • numéro de téléphone ;
  • formation souhaitée ;
  • message libre ;
  • autres informations communiquées volontairement.

3. Finalités

Les données sont utilisées pour :

  • répondre aux demandes envoyées via les formulaires ;
  • gérer les pré-inscriptions et accès prioritaires ;
  • assurer le suivi des personnes intéressées par les formations ;
  • organiser les échanges avec les visiteurs et prospects.

4. Base légale

Le traitement repose sur le consentement de la personne concernée lorsqu’elle remplit un formulaire et accepte d’être recontactée.

5. Destinataires

Les données sont destinées à LOGIX FORMA et aux outils techniques utilisés pour la réception et la gestion des formulaires.

6. Durée de conservation

Les données sont conservées pendant une durée proportionnée à la finalité du traitement, sauf obligation légale contraire.

7. Droits des personnes

Vous disposez d’un droit d’accès, de rectification, d’effacement, de limitation et d’opposition selon la réglementation applicable. Pour exercer vos droits, vous pouvez écrire à : logix.forma@gmail.com.

8. Sécurité

LOGIX FORMA met en œuvre des mesures raisonnables de sécurité pour protéger les données personnelles contre les accès non autorisés, la perte ou l’altération.
